# OuO

## 介紹
    透過 pygame 實作的簡易遊戲。
    我們將生物設計為史萊姆(Slime)，它們具有一定的生命值和食物收集能力，
    在場地中會隨機生成食物 (蘋果)，史萊姆會收集這些食物並帶回巢穴。
    當史萊姆的生命值低於某個門檻值時，它會進入「飢餓」狀態並返回巢穴進食，
    而當史萊姆吃到一定數量時將會產下後代。
    以此進而模擬生物的群體生活模式。
